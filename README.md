🛍️ Customer Behavior Analysis (SQL + Python + Power BI) :

📌 Project Summary

This project focuses on analyzing customer shopping behavior to uncover meaningful insights that can help businesses improve sales, customer engagement, and decision-making. The analysis is performed using Python, SQL, and Power BI.

---

🎯 Business Problem

A retail company wants to better understand its customers’ purchasing behavior across different demographics, product categories, and payment methods.

Key Question:
How can customer shopping data be used to identify trends, improve customer engagement, and optimize business strategies?

---

🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib)
- MySQL
- Power BI
- GitHub

---

📊 Project Workflow

1️⃣ Data Preparation (Python)

- Loaded dataset using Pandas
- Cleaned and preprocessed data
- Handled missing values
- Performed Exploratory Data Analysis (EDA)

2️⃣ Data Storage (MySQL)

- Created database and tables
- Imported cleaned data into MySQL
- Structured data for analysis

3️⃣ Data Analysis (SQL)

- Used SQL queries to extract insights
- Performed aggregations and grouping
- Analyzed revenue, customer segments, and trends

4️⃣ Data Visualization (Power BI)

- Built interactive dashboard
- Visualized key metrics and patterns
- Created charts for better understanding

---

📈 Key Insights

- Male customers contribute higher revenue compared to female customers
- Clothing category has the highest number of purchases
- Most customers prefer weekly or fortnightly purchases
- Payment methods like Credit Card and PayPal are widely used
- Discounts and seasonal trends influence purchasing analysis
---

🚀 How to Run This Project

1. Clone the repository
2. Open Jupyter Notebook and run analysis
3. Connect to MySQL and execute SQL queries
4. Open Power BI file to view dashboard
